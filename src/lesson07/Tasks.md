## Битовый сдвиг
```
        int a = 0xFFFFFFF0;

        int b = a >> 4;//0b011011001;
        int c = a >>> 4;//0b011011001;

        System.out.printf("%X\n",a);
        System.out.printf("a >> 4 = %X\n",b);
        System.out.printf("a >>> 4 = %X\n",c);
```
